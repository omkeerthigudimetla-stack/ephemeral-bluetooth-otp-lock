# Ephemeral Bluetooth OTP Lock

## Description
A smart lock system using Arduino, servo motor, and Bluetooth module (HC-05) with OTP-based authentication via MIT App Inventor.

## Components Used
- Arduino
- Servo Motor
- HC-05 Bluetooth Module

## Working
User enters OTP in mobile app → sent via Bluetooth → Arduino verifies → Servo unlocks
